Why Won't the Dialog Box Accept My Data Entry?


If you have entered data into the dialog box and hit return and are wondering why the data is not accepted, it is because the MATLAB dialog box requires the user to actually click the "OK" button with the mouse or to hit the tab button before hitting return to force the input dialog to set focus on the "OK" button.


This is a "bug" or  "feature" in the MATLAB inputdlg routine.  There are no plans to to try to change the GUI to correct this behavior.
